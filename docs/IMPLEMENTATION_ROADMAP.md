# Implementation Roadmap

## Phase 1, Core product hardening

- Connect custom login UI to auth API.
- Complete Microsoft callback mapping.
- Enable MySQL migrations.
- Add server-side validation for each module.
- Connect module pages to live APIs.
- Add create and edit forms.

## Phase 2, AI working assistant

- Add data retrieval functions per module.
- Add AI source citations.
- Add action approval cards.
- Add prompt history.
- Add admin controls for provider switch.

## Phase 3, Enterprise operations

- Add Teams and Email templates.
- Add SLA engine.
- Add escalation scheduler.
- Add Excel confirm import.
- Add report exports.
- Add full audit reports.

## Phase 4, Production readiness

- Add test coverage.
- Add CI/CD.
- Add monitoring.
- Add production logs.
- Add VAPT fixes.
- Add backup and restore procedure.
